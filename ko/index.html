FOMO 초보 가이드
모델 발행과 참여를 처음부터 이해하기

FOMO(FLock Open Model Offering)는 FLock 생태계의 모델 발행 Launchpad입니다. 발행자는 RMO(Real Model Asset Offering)를 시작할 수 있고, 서포터는 모집 기간에 $FLOCK으로 참여하며, 모델이 Graduate한 뒤 Model Token을 통해 이후 수익 및 Buy-back & Burn 메커니즘에 들어갑니다. 본 가이드는 발행, 참여, 수수료, 모집 결과, 출시 이후 메커니즘 순으로 설명합니다.

01 — WHAT

FOMO란?

FLock은 모델 학습(AI Arena), 연합학습 협업(FL Alliance), API 추론(API Platform)을 이미 갖추고 있습니다. FOMO는 모델 발행 및 자산화 진입점을 메웁니다: 모델을 모집 가능하고, 출시 가능하며, 사용량에 따라 수익을 내는 RMA(Real Model Asset)로 바꿉니다.

한 줄 정의

AI 모델을 위한 Launchpad입니다. 발행자가 RMO 모집을 시작하고 모델을 출시합니다. 모델에 실제 API 호출이 발생하면 수익은 규칙에 따라 Model Token 관련 메커니즘으로 들어갑니다.

어떤 문제를 푸나

모델 역량은 확보할 수 있지만 발행, 모집, 수익 배분은 종종 불명확합니다. FOMO는 API 사용, 모델 수익, Model Token 보유자 간 관계를 실행 가능한 온체인 규칙으로 정리합니다.

생태계 흐름

AI Arena(학습) → FL Alliance(협업) → API Platform(추론) → FOMO(발행 및 MT 메커니즘). 흐름도 펼치기 ↓

생태계 흐름도 · 노드를 클릭하면 해당 섹션으로 이동

기본 접힘 상태로 첫 화면 정보 과부하를 피합니다. 모델 Graduate 이후 수익과 Buy-back & Burn은 §8을 참고하세요.

02 — WHO

어떤 역할인가요?

FOMO는 발행자와 서포터 모두를 위한 플랫폼입니다. 먼저 자신의 역할을 확인한 뒤 해당 경로(경로 A / 경로 B)로 진행하세요.

PATH A

모델 발행자
Real Model Asset (RMA) Issuer

  • 모델을 배포·출시해 API로 호출 가능한 RMA를 만듭니다
  • Launch 페이지에서 RMO(Real Model Asset Offering, 모델 자산 발행)를 시작합니다
  • 고정 발행비(32,768 $FLOCK) 및 관련 거래 수수료를 지불해야 합니다
  • 본 가이드의 「경로 A」에 해당
PATH B

서포터 / 참여자
Users & RMA Supporters

  • Marketplace에서 모집 중인 RMO를 선택하고 $FLOCK으로 참여합니다
  • Bonding Curve 가격으로 해당 모델의 Model Token(MT)을 매수합니다
  • 모델 Graduate 후 MT를 보유하거나 API 사용자로 모델을 호출할 수 있습니다
  • 본 가이드의 「경로 B」에 해당

04 — PATH A · HOW

경로 A: 발행자가 모델을 시작하는 방법

RMA Issuer를 위한 안내입니다. FOMO Launch 페이지에서 진행하며, 추론 가격 티어·모집 구간 등 핵심 파라미터는 플랫폼이 미리 설정해 발행자가 임의로 바꿀 수 없습니다.

1

Launch 페이지로 이동

FOMO Launch(발행) 페이지를 엽니다. RMO를 시작하는 출발점입니다.

2

대상 모델 선택

발행할 모델을 선택합니다. 각 모델에는 미리 정해진 추론 가격 티어와 허용 모집 구간이 대응됩니다.

이 파라미터는 플랫폼이 고정하며 가격·모집 구간을 제한합니다. 임의로 변경할 수 없습니다.
3

자동 입력된 파라미터 확인

모델을 선택하면 관련 값이 해당 모델 기준으로 자동 입력됩니다. 확인 후 진행하면 됩니다 — 역시 임의 변경은 불가합니다.

4

(선택) 첫 토큰 매수 번들

발행자는 초기 매수를 번들로 묶어 Bonding Curve 초기 가격 구간에 최대한 가깝게 맞출 수 있습니다. ↓ 곡선 메커니즘 보기

Bonding Curve는 모집 전 기간 동안 계속 적용됩니다. 매 매수마다 곡선 가격이 변하며, 발행 순간에만 가격이 정해지는 것이 아닙니다.
5

발행 비용 확인 및 승인

RMO를 정식 시작하기 전 발행자는 발행 관련 비용(고정 발행비 포함)을 확인·승인해야 합니다. 비용 상세는 §5를 참고하세요.

6

RMO 시작 후 Marketplace 진입

시작하면 해당 RMO가 Marketplace에 표시되어 공개 모집 단계로 들어갑니다.

7

모집 종료 대기

모집 진행(최소/최대 목표 대비)은 실시간으로 추적됩니다. 모집 종료 전까지 추가 조치는 필요 없습니다(달성/미달성은 §7).

곡선은 오르기만 함

누적 매수가 많을수록 MT 단가는 곡선을 따라 오른쪽 위로 올라갑니다. 모집 수요는 곡선 가격에 바로 반영됩니다.

이른 참여의 이점

일찍 참여할수록 곡선 위치가 보통 낮아 동일 $FLOCK으로 더 많은 MT를 받을 수 있습니다(당시 세율은 차감). 발행자의 첫 매수 번들은 곡선 앞구간에 가깝게 맞추려는 의도입니다.

전 기간 동적 가격

Bonding Curve는 모집 전 기간 동안 계속 적용되며 발행 순간에만 국한되지 않습니다 — $FLOCK 매수마다 가격이 변합니다.

최저가 ≠ 최저 비용

곡선 앞구간 가격은 낮지만 개장 anti-sniping tax는 더 높습니다. 세율이 내려갈 때까지 기다리면 곡선이 이미 올라갔을 수 있습니다 — 진입 시점은 스스로 판단하세요(본 페이지는 투자 조언이 아닙니다).

06 — PATH B · HOW

경로 B: 서포터가 모집에 참여하는 방법

RMA Supporters를 위한 안내입니다. Marketplace에서 RMO를 선택하고 $FLOCK으로 모집에 참여하며, Bonding Curve에 따라 해당 Model Token(MT)을 받습니다.

1

Marketplace 둘러보기

Marketplace에서 진행 중인 RMO를 보고 모델 정보, 가격 티어, 모집 구간을 확인합니다.

2

모델 선택 후 $FLOCK 투입

대상 RMO를 선택해 모집 기간에 $FLOCK을 투입하고 현재 Bonding Curve 가격으로 MT를 매수합니다.

곡선이 앞쪽일수록 MT 단가는 보통 낮지만 개장 anti-sniping tax는 더 높습니다. 진입 시 세율과 곡선 위치를 함께 고려하세요. 곡선 다시 보기 ↑
3

모집 진행 실시간 추적

해당 RMO의 최소/최대 모집 목표 대비 실시간 진행을 추적해 Graduate 가능성을 판단합니다.

4

모집 결과 대기

모집 종료 후 결과에 따라 처리됩니다: 달성 시 모델 출시 후 MT 보유, 미달성 시 $FLOCK 원금 환불(수수료는 환불 안 됨, §7 참고).

08 — LIFECYCLE

모델 출시 이후 일어나는 일

모델 Graduate 후 핵심 변수는 실제 API 호출량입니다. 추론 수익에서 비용을 뺀 뒤 Buy-back & Burn 등 온체인 메커니즘으로 들어갑니다.

수익 메커니즘 · 노드 클릭 또는 호버로 고정

사용이 곧 수익

사용자가 FLock API Platform으로 모델을 유료 호출할 때마다 추론 수익이 발생합니다. 모델 출시 후 실제 호출량이 성과를 보는 핵심 지표입니다.

다이어그램은 자동 재생됩니다. 호버하거나 노드를 클릭하면 멈춰 볼 수 있습니다. 아래 세 카드는 각 단계 보충 설명입니다.

사용이 곧 수익

API Platform 유료 호출로 실제 추론 수익이 생깁니다. 출시 후에는 모집 단계 데이터만이 아니라 실제 호출량에 집중하세요.

Buy-back & 소각

추론 순수익은 규칙에 따라 온체인에서 해당 MT를 Buy-back & Burn하며, 모델 사용 가치를 MT 보유자 장기 메커니즘에 반영합니다.

스테이킹으로 할인

보유자는 MT를 스테이킹(락업)해 API 호출 요율을 낮출 수 있습니다. 발행자도 추론 비용을 보조해 초기 API 사용자를 끌어들일 수 있습니다.

2차 시장 유동성: Graduate 후 Model Token은 Aerodrome 등 유동성 허브에서 거래할 수 있습니다.

요약: 발행자의 일회성 고정 비용입니다. 모델 배포, 온체인 설정, 발행 프로세스를 포함하며 모든 RMO에 동일해 사전에 계산하기 쉽습니다.

「anti-sniping tax」는 무엇인가요?

요약: 개장 직후 sniping을 억제합니다. 종합 세율은 99%에서 시작해 매분 1%씩 내려가 기본 수수료까지 돌아가며, 봇·스나이퍼의 저비용 순간 매수를 어렵게 합니다. ▶ 99% 감쇠 애니메이션

일찍 살수록 항상 유리한가요?

요약: 꼭 그렇지는 않습니다. Bonding Curve 앞쪽은 가격이 낮지만 개장 anti-sniping tax가 더 높고, 세율이 내려갈 때까지 기다리면 곡선이 올라갔을 수 있습니다. 둘을 함께 고려해야 하며 본 페이지는 투자 조언이 아닙니다. 자세히는 §05.5.

모집에 실패하면 무엇을 잃나요?

요약: 서포터는 $FLOCK 원금을 돌려받지만 거래 수수료는 환불되지 않습니다. 발행자가 낸 발행비도 환불되지 않습니다. §7 슬라이더를 움직여 두 결과를 비교하세요.

Model Token과 $FLOCK의 관계는?

요약: 서로 다릅니다. $FLOCK은 FLock 네이티브 토큰으로 RMO 모집 참여 및 발행 비용 결제에 쓰이고, Model Token(MT)은 개별 RMA 전용 토큰으로 모델 하나당 MT 하나입니다.
